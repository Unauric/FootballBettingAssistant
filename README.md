# Football Betting Assistant

A data analysis project that uses historical football match data to generate insights and assist with informed betting decisions.

## Overview

Football Betting Assistant processes multiple seasons of football match data to analyze team performance, trends, and statistical patterns. The goal is to provide structured insights rather than automated betting, making it suitable for analytical and educational use.

## Features

- Multi-season football match analysis
- Team performance and form evaluation
- Statistical trend extraction
- CSV-based data processing
- Extendable for predictive modeling

## Data

The project includes historical datasets covering multiple seasons:

- `E2020.csv`
- `E2021.csv`
- `E2022.csv`
- `E2023.csv`
- `E2024.csv`
- `E2025.csv`

Each file contains structured match results, scores, and team statistics.

## Use Cases

- Football data analysis
- Betting strategy research
- Sports analytics practice
- Data science portfolio projects
